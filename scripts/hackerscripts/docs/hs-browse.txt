DESCRIPTION
-----------

hs-browse

Opens all the URLs are specified in the configuration with the
default web browser available.

CONFIGURATION
-------------

Here's an example configuration:

[hs-browse]
	url1 = www.github.com
	url2 = www.stackoverflow.com
	url3 = www.facebook.com
	url4 = mail.google.com

1. URLs

The URLs.

USAGE
-----

Check hs-browse -h
