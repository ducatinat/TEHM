DESCRIPTION
-----------

hs-desktop

Got un-organized files all over your desktop? Run hs-desktop to move all of
them in their respective directories.

CONFIGURATION
-------------

Here's an example configuration:

[hs-desktop]
	files_directory = C:\Users\Areeb\Desktop\Files
	images_directory = D:\Media\Images
	videos_directory = D:\Media\Videos

1. FILES_DIRECTORY

Directory to move your files to. 

Supported file types:
		".txt", ".ppt", ".doc", ".xls"

2. IMAGES_DIRECTORY

Directory to move your images to.

Supported file types:
		".png", ".jpg", ".gif"

3. VIDEOS_DIRECTORY

Directory to move your videos to.

Supported file types:
		".avi", ".mp4", ".mkv", ".flv", ".wmv", 
		".m4v", ".mpg", ".3gp", ".3g2", ".f4v"

USAGE
-----

hs-desktop [-h]

Required arguments:

None

Optional arguments:

--h, -h		Display this help documentation
